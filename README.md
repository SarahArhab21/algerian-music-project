# 🧠 Algerian Artists Web Application
The Algerian Artists Web Application is a web-based platform that showcases Algerian artists, their biographies, and their work. The application is built using the Flask framework and utilizes XML data to store information about the artists. The platform provides a user-friendly interface for searching and exploring artists, and it generates biographies for each artist based on their name, city, and style.

## 🚀 Features
- Search for artists by name
- Generate biographies for artists
- Render HTML templates using parsed XML data
- Handle requests to the root URL and render the index HTML template
- Parse XML file containing artist data
- Integrate with other files, such as XSL files, to transform XML data into HTML

## 🛠️ Tech Stack
* Frontend: HTML, CSS
* Backend: Python, Flask
* Database: XML
* AI Tools: None
* Build Tools: None
* Dependencies:
  - Flask: a micro web framework for Python
  - lxml: a library for parsing and manipulating XML and HTML documents
  - os: a built-in Python library for interacting with the operating system

## 📦 Installation
To install the application, follow these steps:
1. Clone the repository using `git clone`
2. Install the required dependencies using `pip install -r requirements.txt`
3. Run the application using `python ArtistAlgerian.py`

## 💻 Usage
To use the application, follow these steps:
1. Open a web browser and navigate to the root URL of the application
2. Search for artists by name using the search bar
3. Click on an artist to view their biography and work
